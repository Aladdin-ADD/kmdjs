##kmdjs是什么？
kmdjs是js工程化终极解决方案

kmdjs基于namespace和class组织项目

kmdjs以统一AMD和CMD为己任

kmdjs以Kill AMD and CMD为目标

故，得名：kmdjs

##Ket是什么？

Ket - Kmdjs Extension Tools

  * Util - :代表公共库，类似Enterprise  Library之类的公共基础库，都是纯粹的Class Library，没有UI元素
  * Base - :组织通用的框架，基于Util之上，面向某个开发领域补充的Class Library和控件

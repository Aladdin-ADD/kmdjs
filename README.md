##kmdjs是什么？
kmdjs是js工程化终极解决方案

kmdjs基于namespace和class组织项目

kmdjs以统一AMD和CMD为己任

kmdjs以Kill AMD and CMD为目标

故，得名：kmdjs

入门文档见: [https://github.com/kmdjs/kmdjs/issues/](https://github.com/kmdjs/kmdjs/issues/)


##kmdjs优势是什么？

支持依赖可视

支持循环依赖

支持命名重复

支持压缩打包

支持代码美化

支持远程加载

支持延迟加载

支持模块共享

支持平铺依赖

支持断点调试

支持独立打包

支持一键下载
